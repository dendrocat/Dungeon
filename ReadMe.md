# Описание

The Rift - это шутер от первого лица с механиками стелса.

## Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| **Стелс-механики** | Проходите уровни незаметно, избегая обнаружения |
| **Умные противники** | Противники управляются с помощью конечного автомата (FSM) |
| **Нейросеть AI** | FSM контролируется нейросетью, обученной специально для этого проекта с помощью фреймворка [ML Agents](https://docs.unity3d.com/Packages/com.unity.ml-agents@4.0/manual/index.html) |
| **Разнообразное оружие** | 5+ видов оружия с уникальными анимациями |

## Цели игры

- Пройти все уровни
- Собрать все улучшения

## Технологии

- **Движок**: Unity
- **Язык**: C#
- **AI**: Unity ML-Agents (нейросеть + FSM)
- **Документация**: Doxygen
